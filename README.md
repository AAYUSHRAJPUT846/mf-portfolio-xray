# MF Portfolio X-Ray
### ET AI Hackathon 2026 — PS9: AI Money Mentor

An AI-powered mutual fund portfolio analyser built for every Indian investor. Upload your CAMS statement and get a complete X-Ray of your portfolio in seconds — for free.

## Features
- Upload CAMS / KFintech PDF statement
- AI-powered XIRR calculation
- Portfolio overlap detection
- Expense ratio drag analysis
- Personalised rebalancing recommendation

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Chart.js
- AI: Groq API (Llama 3.3 70B) — free tier
- PDF Parsing: pdf.js
- Hosting: Replit
